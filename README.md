 # KeenKeeper



## 🎯 Description

Managing friendships in today's busy world can be challenging. KeenKeeper makes it simple by providing a centralized hub to:
- Track all your interactions (calls, texts, videos)
- Monitor when you last contacted each friend
- Set and maintain relationship goals
- View analytics on your friendship patterns


## 🛠️ Technologies Used

- **Frontend Framework**: React js
- **Routing**: React Router DOM  
- **Styling**: Tailwind CSS + DaisyUI
- **Charts & Analytics**: Recharts
- **Notifications**: React Toastify
- **Build Tool**: Vite
- **Icons**: React Icons
- **Package Manager**: npm
